# CNA SQL Projects

A collection of my SQL portfolio projects focused on data cleaning, exploratory analysis, and relational database insights. These projects demonstrate practical SQL skills used in real-world analytics workflows, including data transformation, joins, aggregations, feature engineering, and business insight generation.

## Projects

### 1. International Layoffs SQL Analysis
Analyzes global layoffs during the COVID period using SQL. This project includes data cleaning, standardization, and exploratory analysis to uncover trends across companies, industries, countries, and funding stages.

**Skills demonstrated:**
- Data cleaning and preprocessing
- Window functions
- Aggregations and grouping
- Trend analysis
- Business insight generation

**Project folder:** `international-layoffs-sql-analysis`

---

### 2. Movies Data Cleaning
Cleans and transforms a messy movie dataset into an analysis-ready format using SQL. The project focuses on duplicate removal, null handling, string cleaning, feature engineering, and data type standardization.

**Skills demonstrated:**
- Data cleaning
- Feature engineering
- String manipulation
- Regular expressions
- Data type conversion

**Project folder:** `movies_data_cleaning`

---

### 3. Restaurant Relational Analysis SQL
Analyzes restaurant menu and order data using a relational database approach. This project combines multiple tables to evaluate ordering behavior, item performance, category trends, and revenue-driving patterns.

**Skills demonstrated:**
- Relational database analysis
- SQL joins
- Aggregations
- Common Table Expressions (CTEs)
- Sales and menu performance analysis

**Project folder:** `restaurant_relational_analysis_sql`
